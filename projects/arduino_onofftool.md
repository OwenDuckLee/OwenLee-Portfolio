# Project Description
## Sovle what problems
- reduce RD testing time
- 循環測試，抓裝置遺失次數
- 可視化的選項介面
- 模式選單
- control AC power on/off

## Tool Featrues
- 優化測試效率 提升 測試精準 度
- 面對各 種 issue 避免 缺乏 測試工具
- 低成本 高維護性 高擴展性
- 無人自動 化 測試 / 自動錯誤紀錄
- 客製化 / 模組化 軟體程式
- 模組化 / 擴充式 治具
- 一鍵搞 定 999 次 / 即時精準 log
- 可依據不同情境 設定環境 參數
- 可隨時 替換 or 擴充 各部位組件
